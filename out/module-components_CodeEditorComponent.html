Module: components/CodeEditorComponent

Vue-Component to display the code and comments together. Depends on vue-codemirror, codemirror and side-comments. Adds Hooks for side-comments to codemirror. Handles display and interaction of and with the comments. Handles display of the code. Holds the configuration for codemirror.
